# Sistema-de-Ordenacao-e-Busca

Um projeto desenvolvido para um trabalho de faculdade.

Este projeto implementa um sistema completo de organização de componentes utilizando algoritmos de ordenação (Bubble Sort, Insertion Sort e Selection Sort) e busca binária.
Desenvolvido em linguagem C, o programa permite cadastrar itens, ordená-los por diferentes critérios e localizar o componente-chave necessário para ativar a “Torre de Fuga”.
O objetivo do trabalho é praticar modularização, análise de desempenho, manipulação de strings e estruturas de dados.
